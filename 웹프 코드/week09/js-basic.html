자바스크립트 기초

1. 자바스크립트 출력

자바스크립트에서 출력하는 여러 가지 방법을 알아봅니다.

2. 변수와 데이터 타입

자바스크립트의 변수 선언과 기본 데이터 타입을 알아봅니다.

3. 연산자

자바스크립트의 기본 연산자를 알아봅니다.

4. 조건문

if-else 조건문을 사용하여 조건에 따라 다른 결과를 출력합니다.

5. 반복문

for 반복문과 while 반복문을 사용하여 반복 작업을 수행합니다.

6. 함수

자바스크립트 함수를 정의하고 호출하는 방법을 알아봅니다.

7. 이벤트

자바스크립트 이벤트를 사용하여 사용자 상호작용에 반응하는 방법을 알아봅니다.
