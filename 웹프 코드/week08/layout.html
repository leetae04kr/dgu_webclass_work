CSS 레이아웃

디스플레이(display) 속성

요소가 화면에 어떻게 표시될지를 결정하는 속성입니다.

display: inline 인라인 요소는 줄바꿈 없이 한 줄에 표시됩니다.
display: block
블록 요소는 항상 새로운 줄에서 시작하며, 가로 영역을 모두 차지합니다.
display: inline-block
인라인-블록 요소는 인라인처럼 한 줄에 표시되지만, 블록처럼 크기를 지정할 수 있습니다.
display: none (이 요소는 보이지 않습니다)

가시성(visibility) 속성

요소를 보이게 하거나 보이지 않게 하는 속성입니다.

visibility: visible (이 요소는 보입니다)
visibility: hidden (이 요소는 보이지 않지만 공간은 차지합니다)

플로트(float) 속성

요소를 왼쪽이나 오른쪽으로 배치하는 속성입니다.

float: left
float: right

플로트 속성을 사용하면 요소를 왼쪽이나 오른쪽으로 배치할 수 있습니다. 텍스트는 플로트 요소 주위를 감싸게 됩니다.

clear 속성을 사용하면 플로트 요소 아래에 다른 요소를 배치할 수 있습니다.

포지션(position) 속성

요소의 위치를 지정하는 속성입니다.

position: static (기본값)
position: relative (상대 위치)
position: absolute (절대 위치)
position: fixed (고정 위치)
