반응형 웹 디자인이란?

반응형 웹 디자인은 다양한 화면 크기와 기기에 맞게 웹 페이지가 자동으로 조정되는 디자인 방식입니다.

미디어 쿼리, 유동적 그리드, 유동적 이미지 등의 기술을 사용하여 구현합니다.

반응형 이미지

max-width: 100%와 height: auto를 사용하여 이미지가 컨테이너 너비에 맞게 조정되도록 합니다.

브라우저 창의 크기를 조절해보세요. 화면 너비에 따라 박스의 배치가 변경됩니다:

  • 768px 이상: 한 줄에 3개의 박스
  • 480px ~ 768px: 한 줄에 2개의 박스
  • 480px 이하: 한 줄에 1개의 박스

뷰포트 단위

vw, vh, vmin, vmax 등의 뷰포트 단위를 사용하면 화면 크기에 비례하여 요소의 크기를 지정할 수 있습니다.

너비: 50vw (뷰포트 너비의 50%)
너비: 50vmin, 높이: 20vmin (뷰포트의 작은 치수 기준)

반응형 웹 디자인의 장점

  • 하나의 웹 사이트로 모든 기기에 대응 가능
  • 별도의 모바일 사이트 개발 불필요
  • 유지 보수 용이
  • 사용자 경험 향상
  • 검색 엔진 최적화(SEO) 향상
