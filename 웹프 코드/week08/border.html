테두리 스타일

border 속성을 사용하여 요소의 테두리 스타일을 지정할 수 있습니다.

solid
dotted
dashed
double

방향별 테두리 스타일

border-top, border-right, border-bottom, border-left 속성을 사용하여 각 방향별로 다른 테두리 스타일을 지정할 수 있습니다.

방향별 테두리

둥근 테두리

border-radius 속성을 사용하여 테두리를 둥글게 만들 수 있습니다.

border-radius: 20px
border-radius: 50%
border-radius: 10px 30px 50px 70px

그림자 효과

box-shadow 속성을 사용하여 요소에 그림자 효과를 추가할 수 있습니다.

box-shadow: 5px 5px 10px #888
box-shadow: 5px 5px 10px 5px #888
box-shadow: 0 0 20px #00f
box-shadow: inset 5px 5px 10px #888
