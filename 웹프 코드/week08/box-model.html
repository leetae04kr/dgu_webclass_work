CSS 박스 모델

박스 모델 개념

웹 문서의 내용은 박스 형태로 정의되며, 이것을 '박스 모델'이라고 합니다.

박스 모델은 콘텐츠(content), 패딩(padding), 테두리(border), 마진(margin)으로 구성됩니다.

박스 모델 예제

박스 모델의 기본 구성 요소

width, height: 콘텐츠 영역의 크기

padding: 콘텐츠와 테두리 사이의 여백

border: 박스의 테두리

margin: 박스 모델 사이의 여백

박스 모델의 전체 너비 계산

전체 너비 = width + left padding + right padding + left border + right border + left margin + right margin

테두리 스타일

solid
dotted
dashed
double
groove
ridge
inset
outset

둥근 테두리

border-radius 속성을 사용하면 테두리를 둥글게 만들 수 있습니다.

이 박스는 border-radius: 20px를 적용했습니다.
