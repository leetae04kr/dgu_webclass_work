HTML과 첫 만남

HTML은 웹에서 자유롭게 오갈 수 있는 웹 문서를 만드는 언어입니다.

텍스트뿐만 아니라 이미지, 영상 등을 표시할 수 있으며, 마크업에 사용하는 꼬리표를 '태그(tag)'라고 합니다.

HTML 구조 파악하기

HTML 문서의 기본 구조

HTML 문서는 다음과 같은 기본 구조를 가집니다:

  • <!DOCTYPE html> - 웹 브라우저에게 HTML 문서임을 알려줍니다.
  • <html> - 웹 문서의 시작과 끝을 나타냅니다.
  • <head> - 문서 관련 정보를 입력합니다.
  • <body> - 실제 브라우저에 표시될 내용을 입력합니다.

웹 문서 구조를 만드는 시맨틱 태그

시맨틱(semantic)은 '의미론적인', '의미가 통하는'이라는 뜻으로, 이름만 봐도 의미를 알 수 있는 HTML 태그를 말합니다.

주요 시맨틱 태그

<header>
헤더 영역을 나타내는 태그로, 사이트 전체의 헤더나 특정 영역의 헤더에 사용합니다.
<nav>
내비게이션 영역을 나타내는 태그로, 웹 문서 안의 다른 위치나 다른 웹 문서로 연결하는 링크를 만듭니다.
<main>
웹 문서에서 핵심이 되는 내용을 담는 태그로, 웹 문서마다 다르게 보여주는 내용으로 구성됩니다.
<article>
독립적인 콘텐츠를 담는 태그로, 따로 떼어도 콘텐츠가 되는 내용을 담습니다.
<section>
콘텐츠 영역을 나타내는 태그로, 몇 개의 콘텐츠를 묶는 용도로 사용합니다.
<aside>
사이드 바 영역을 나타내는 태그로, 본문 내용 외에 왼쪽이나 오른쪽, 혹은 아래쪽에 사이드 바를 표시합니다.
<footer>
푸터 영역을 나타내는 태그로, 사이트 제작 정보나 저작권 정보, 연락처 등을 포함합니다.
<div>
여러 소스를 묶는 태그로, 영역을 구별하거나 스타일을 적용하기 위해 사용합니다.
