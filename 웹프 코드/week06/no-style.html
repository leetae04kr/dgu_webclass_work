CSS 기본 스타일

스타일과 스타일 시트

CSS는 Cascading Style Sheets의 약자로, HTML 문서의 디자인을 담당합니다.

HTML로는 웹 사이트의 내용을 나열하고 CSS로는 웹 문서의 디자인을 구성합니다.

스타일 형식 알아보기

스타일 규칙은 선택자(selector)와 선언부(declaration)로 구성됩니다.

선택자는 스타일을 적용할 대상을 지정하고, 선언부는 스타일 내용을 정의합니다.

스타일 시트의 종류

  • 인라인 스타일 - 태그에 직접 style 속성 사용
  • 내부 스타일 시트 - head 태그 안에 style 태그로 정의
  • 외부 스타일 시트 - 별도의 CSS 파일로 저장하여 연결

CSS 기본 선택자

  • 전체 선택자(*) - 모든 요소에 스타일 적용
  • 타입 선택자 - 특정 태그를 사용한 모든 요소에 스타일 적용
  • class 선택자(.) - 특정 부분에만 스타일 적용, 문서 안에서 여러 번 사용 가능
  • id 선택자(#) - 특정 부분에만 스타일 적용, 문서 안에서 한 번만 사용
  • 그룹 선택자 - 여러 선택자에 같은 스타일을 적용할 때 사용

캐스케이딩 스타일 시트

캐스케이딩(Cascading)은 '위에서 아래로 흐른다'는 뜻으로, 스타일이 계단식으로 적용된다는 의미입니다.

스타일 충돌을 막기 위한 원칙:

  1. 스타일 우선순위 - 중요도와 명시도에 따라 결정
  2. 스타일 상속 - 부모 요소의 스타일이 자식 요소로 전달
