텍스트 스타일 적용하기

글꼴 관련 스타일

font-family 속성은 웹 문서에서 사용할 글꼴을 지정합니다.

font-size 속성은 글자 크기를 지정합니다.

font-style 속성은 글자를 이탤릭체로 표시합니다.

font-weight 속성은 글자 굵기를 조절합니다.

웹 폰트 사용하기

웹 폰트는 웹 문서 안에 글꼴 정보도 함께 저장했다가 사용자가 웹 문서에 접속하면 글꼴을 사용자 시스템으로 다운로드시켜 사용하는 글꼴입니다.

이 페이지의 제목은 구글 웹 폰트 중 'Nanum Pen Script'를 사용하고 있습니다.

텍스트 관련 스타일

color 속성은 글자 색을 지정합니다.

text-align 속성은 텍스트 정렬 방법을 지정합니다.

line-height 속성은 줄 간격을 지정합니다.

text-decoration 속성은 텍스트에 밑줄이나 취소선을 표시합니다.

text-shadow 속성은 텍스트에 그림자 효과를 추가합니다.
